# Contributing to nwf-vision

See [nwf-core CONTRIBUTING](https://github.com/romero19912017-ui/nwf-core/blob/main/CONTRIBUTING.md) for general guidelines.

Additional requirements for nwf-vision:

- Tests require torch and torchvision
- Examples use CIFAR-10 (auto-download)
- Run: `pytest tests -v` (ensure nwf-core is installed)
